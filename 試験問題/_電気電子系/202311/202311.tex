\item
次の微分方程式で表されるシステムを考える．
\[
	 \ddot{\theta} = a \sin \theta + b (\dot{\theta})^2 + (\cos \theta)u
\]
ここで$\theta$は機械系のリンク角に相当するもの，$u$は入力，$a,b$は定数とする．

\begin{enumerate}
\item 
このシステムを$(\theta, \dot{\theta})^T$を状態とする非線形状態方程式であらわせ．
\vspace{6cm}

\item
状態方程式をテーラー展開の１次近似線形化せよ．
\vspace{10cm}

\item
微分方程式を厳密に線形化するフィードバックを設計し，厳密に線形化された状態方程式を示せ．ただし，$-\pi/2 < \theta < \pi/2$と仮定する．
\vspace{5cm}

\end{enumerate}
