\item 
次のシステム
\begin{align}
	\frac{dx}{dt}=Ax+Bu
	, \ \ \ \ \ 
	y= Cx
\nonumber
\end{align}
に対して状態フィードバックとオブザーバ
\begin{align}
	u = Fx, \ \ \ \ \ 
	\frac{d \hat{x}}{dt} = A \hat{x} + B u - K (y-C\hat{x})
\nonumber
\end{align}
を$A+BF$と$A+KC$の固有値の実部が負となるように設計したとする．元のシステムを次のコントローラ
\begin{align}
	u = F\hat{x}, \ \ \ \ \ 
	\frac{d \hat{x}}{dt} = A \hat{x} + B u - K (y-C\hat{x})
\nonumber
\end{align}
で制御するとき，以下の問に答えよ．
%================================================================

\begin{enumerate}
\item 
元のシステムとコントローラの閉ループ系の状態方程式を$\left( \begin{array}{c} x \\ \hat{x} \end{array} \right)$を状態として求めよ．
\vspace{7cm}

\item
閉ループ系の状態を$\bmatrix{c} x \\ \hat{x} \ematrix$から$\bmatrix{c} x \\ e \ematrix$, $ (e= \hat{x}- x) $に変換する座標変換行列を求めよ．
\vspace{3cm}

\item
閉ループ系の状態方程式を$\bmatrix{c} x \\ e \ematrix$を状態として表せ．
\vspace{10cm}

\item
これらの結果より閉ループ系の極についてわかることについて述べよ．
\end{enumerate}
