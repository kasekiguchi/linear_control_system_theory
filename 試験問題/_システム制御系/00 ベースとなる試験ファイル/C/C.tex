\item
状態方程式
\[
	\frac{dx}{dt}= Ax
\]
の$A$が以下のリアプノフ方程式を満すと仮定する．以下の問に答えよ．
\[
	PA + A^T P = -Q
\]
\begin{enumerate}
\item 
$P$, $Q$が正定（$P>0$, $Q>0$）である場合に$t \rightarrow \infty$で$x \rightarrow 0$となることを，リアプノフ関数$V=x^T P x$を時間微分することにより説明せよ．
\vspace{9cm}

\item
$P$は正定（$P>0$）であるが，$Q$は半正定であり，$Q=H^T H \geq 0$, $H \in R^{p \times n}$, $p < n$と表される場合，$t \rightarrow \infty$で$x \rightarrow 0$となるとは限らない．その理由を説明せよ．
\vspace{6cm}

\item
$(H,A)$可観測の場合には$t \rightarrow \infty$で$x \rightarrow 0$となる．その理由を説明せよ．
\end{enumerate}
