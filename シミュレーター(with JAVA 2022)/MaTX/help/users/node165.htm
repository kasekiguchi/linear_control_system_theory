有理多項式行列の評価

  関数eval()を用いて，有理多項式行列の変数にいろいろなクラスの値を 代入した結果を求めることができる。 配列は，有理多項式行列の各成分の変数に代入されるので， 評価結果はブロック行列になる。 例えば，変数に行列を代入して 
s = $;
RM1 = [[(s+1)/(s+2)  (s+3)/(s+4)]
       [(s+5)/(s+6)  (s+7)/(s+8)]];

A = [[1 2][3 4]];

B = eval(RM1, A);

の結果，


となるが，配列を用いると 
s = $;
RM1 = [[(s+1)/(s+2)  (s+3)/(s+4)]
       [(s+5)/(s+6)  (s+7)/(s+8)]];

A = [[1 2][3 4]];

B = eval(RM1, Array(A));

の結果，


となる。有理多項式行列の評価に伴うクラス変換の規則を以下に示す。
