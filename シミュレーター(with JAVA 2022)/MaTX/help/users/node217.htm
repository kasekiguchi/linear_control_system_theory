複素共役転置 (Complex Conjugate Transpose)

   整数，実数，実(多項式，有理多項式)は複素共役転置によって変わらない。 実(行列，配列)の複素共役転置は，転置と同じである。

整数=整数' = 整数#
実数=実数' = 実数#
複素数=複素数#
実多項式=実多項式'= 実多項式#
複素多項式=複素多項式#
実有理多項式=実有理多項式'= 実有理多項式#
複素有理多項式=複素有理多項式#
実(行列|配列)=実(行列|配列)' = 実(行列|配列)#
複素(行列|配列)=複素(行列|配列)#
実多項式(行列|配列)=実多項式(行列|配列)'
 =実多項式(行列|配列)#
複素多項式(行列|配列)=複素多項式(行列|配列)#
実有理多項式(行列|配列)=実有理多項式(行列|配列)'
 =実有理多項式(行列|配列)#
複素有理多項式(行列|配列)=複素有理多項式(行列|配列)#
