1で満たされた行列

多項式行列の場合，成分の全てが定数項が1である多項式になる。 有理多項式行列の場合，成分の全てが分子多項式と分母多項式の定数項が1である 有理多項式になる。 ピース行列を大きさの単位として使い， ピース行列の整数倍の大きな1で満たされた行列を簡単に作ることができる。     

M1 = ONE(3);        3x3の実行列
M1 = ONE(3, c);     3x3の複素行列,         c = 複素数
M1 = ONE(3, P);     3x3の実多項式行列,     P = 実多項式
M1 = ONE(3, CP);    3x3の複素多項式行列,   CP = 複素多項式
M1 = ONE(3, R);     3x3の実有理多項式行列,   R = 実有理多項式
M1 = ONE(3, CR);    3x3の複素有理多項式行列, CR = 複素有理多項式
M1 = ONE(3, 4);     3x4の実行列,
M1 = ONE(3, 4, c);  3x4の複素行列,         c = 複素数
M1 = ONE(3, 4, P);  3x4の実多項式行列,     P = 実多項式
M1 = ONE(3, 4, CP); 3x4の複素多項式行列,   CP = 複素多項式
M1 = ONE(3, 4, R);  3x4の実有理多項式行列,   R = 実有理多項式
M1 = ONE(3, 4, CR); 3x4の複素有理多項式行列, CR = 複素有理多項式
M2 = ONE(M1);       M1と同じ大きさで同じクラスの行列
M2 = ONE(3, M1);    M1を横に3個並べた大きさで同じクラスの行列
M2 = ONE(2, 3, M1); M1を縦に2個,横に3個並べた大きさで
                    M1と同じクラスの行列
