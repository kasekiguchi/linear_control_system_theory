行ベクトル

成分をスペース，タブ，改行，コンマ ``,'' のいずれかで区切って書き並べ， 四角ブラケット [ と ] で囲む。 
M1 = [];              // 0 x 0の行列
M1 = [1 2 3];         // スペースで区切る
M1 = [1 2 , -3];      // -単項演算子をカンマで区切る
M1 = [1 2 (-3)];      // -単項演算子を括弧で囲む
CM1 = [(1,2), (3,4)]; // 複素行ベクトル
