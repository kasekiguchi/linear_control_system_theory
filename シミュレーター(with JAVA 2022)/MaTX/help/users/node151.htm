零行列

    ピース行列15.1を大きさの単位として使い， ピース行列の整数倍の大きな零行列を簡単に作ることができる。 

M1 = Z(3);        3x3の実零行列
M1 = Z(3, c);     3x3の複素零行列,         c = 複素数
M1 = Z(3, P);     3x3の実多項式零行列,     P = 実多項式
M1 = Z(3, CP);    3x3の複素多項式零行列,   CP = 複素多項式
M1 = Z(3, R);     3x3の実有理多項式零行列,   R = 実有理多項式
M1 = Z(3, CR);    3x3の複素有理多項式零行列, CR = 複素有理多項式
M1 = Z(3, 4);     3x4の実零行列,
M1 = Z(3, 4, c);  3x4の複素零行列,         c = 複素数
M1 = Z(3, 4, P);  3x4の実多項式零行列,     P = 実多項式
M1 = Z(3, 4, CP); 3x4の複素多項式零行列,   CP = 複素多項式
M1 = Z(3, 4, R);  3x4の実有理多項式零行列,   R = 実有理多項式
M1 = Z(3, 4, CR); 3x4の複素有理多項式零行列, CR = 複素有理多項式
M2 = Z(M1);       M1と同じ大きさで同じクラスの零行列
M2 = Z(3, M1);    M1を横3個並べた大きさで同じクラスの零行列
M2 = Z(2, 3, M1); M1を縦2個に横に3個並べた大きさで
                  M1と同じクラスの零行列)
