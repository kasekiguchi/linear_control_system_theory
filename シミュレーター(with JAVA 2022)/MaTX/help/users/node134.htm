有理多項式

実数と複素数を係数とする1変数の有理多項式の数式処理を行なうことができる。

  ［宣言］ 

  Rational   a, b, c;     // 実有理多項式
  CoRational a2, b2, c2;  // 複素有理多項式

［例］ 

  (s + 2)/(2*s + 3),  (1,2)/((3,4)*s + (5,6))
