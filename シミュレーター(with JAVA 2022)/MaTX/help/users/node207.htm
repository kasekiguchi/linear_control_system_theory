メモリの状態遷移

メモリの状態遷移の様子を次の例題を使って示す。 
Func Matrix afo()
{
  Matrix A, B, C, D, E;

  A = [[1 2][3 4]];      // (1)
  B = [[5 6][7 8]];      // (2)
  C = A + B;             // (3)
  D = (A + B) * (A - B); // (4)
  E = (C + D) * (C - D); // (5)
  return E;              // (6)
}

式(5)の右辺が評価されたときのメモリの状態は 図 B.10のようになっている。d, e, gには， それぞれ(C + D), (C - D), (C + D)*(C - D)の計算結果が入っている。 dとeはREV状態からgはSYS状態から TMP状態に移った。
