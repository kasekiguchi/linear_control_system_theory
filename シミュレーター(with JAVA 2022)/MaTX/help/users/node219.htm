{ポスト,プリ}{インクリメント,デクリメント}

   MATXには変数のインクリメントとデクリメントに対して，C言語のそれと同様な 2つの演算子がある。 インクリメント演算子 ++ はその被演算数に1を加え， デクリメント演算子 - は1を引くものである。 演算子 ++ と - は前置演算子と後置演算子のどちらにも使える。 

  整数++     // 整数 = 整数 + 1
  実数++     // 実数 = 実数 + 1.0
  整数--     // 整数 = 整数 - 1
  実数--     // 実数 = 実数 - 1.0
  ++整数     // 整数 = 整数 + 1
  ++実数     // 実数 = 実数 + 1.0
  --整数     // 整数 = 整数 - 1
  --実数     // 実数 = 実数 - 1.0
