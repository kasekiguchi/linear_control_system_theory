関数の参照

MATXでは，関数それ自身は変数ではないが， 関数の参照を関数の引数として渡すことができる。 関数の参照は，関数の名前と同じように使える。 ただし，C言語の関数のポインタに対応するような記述方法はない。   
Func void main()
{
  Matrix a;
  void foo(), bar();

  a = [[1 2][3 4]];
  foo(bar, a);
}

Func void foo(afo, a)
  void afo();
  Matrix a;
{
  afo(a);
}

Func void bar(a)
  Matrix a;
{
  print a;
}
