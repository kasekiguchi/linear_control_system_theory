Do-whileループ

  do-whileループでは，ループ本体を通る各パスを終ったあとにテストが 行なわれる。ループ本体はつねに，少なくとも１回は実行される。その構文は 
  do {
      statements;
  } while (expr) ;

ここでは，まずstatementsが実行された後，exprが計算される。 それが真ならば，statementsは再び実行される。 このループが終了するのは，exprが偽になったときである。
