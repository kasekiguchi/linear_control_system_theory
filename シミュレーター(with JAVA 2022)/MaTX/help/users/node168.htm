有理多項式配列の分子配列と分母配列

    有理多項式配列の各要素の分子多項式からなる分子配列と 分母多項式からなる分母配列は， 関数Nu()とDe()を用い取り出す。 
  s = $;
  A = [[1 2]     B = [[5 6]
       [3 4]];        [7 8]];

  C = (ONE(2)*s + A) ./ (ONE(2)*s + B);

  // C = [[(s+1)/(s+5)  (s+2)/(s+6)]
  //      [(s+3)/(s+7)  (s+4)/(s+8)]]

  N = Nu(Array(C)); // N = [[s+1  s+2]
                    //      [s+3  s+4]]

  D = De(Array(C)); // D = [[s+5  s+6]
                    //      [s+7  s+8]]
