文字列から指定文字の位置を調べる

    ある文字列中に指定した1文字が存在するか，存在するなら何文字目であるか を関数strchr()にって調べられる。関数strchr()の第一引数は 対象となる文字列，第二引数は調べたい1文字を指定する。もし，対象文字列に 指定した文字が存在しない場合，関数strchr()は0を返す。 例えば， 
  str = "Hello world";
  id1 = strchr(str, "w");
  id2 = strchr(str, "A");

を実行すると，id1 = 7，id2 = 0の結果を得る。
