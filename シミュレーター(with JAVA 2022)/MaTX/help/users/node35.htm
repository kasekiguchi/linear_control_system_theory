ヘルプ

  helpコマンドを実行すると関数一覧が表示される。 help topicを実行すると，指定したトピックスに関する情報が 表示される。topicがヘルプファイル中で見つからない場合は， topic.mmというmmファイルが探される。そして，そのファイルが存在 した場合，ファイルの先頭部分のコメント(/*と*/で囲まれた部分) が画面に表示される。ヘルプメッセージが表示されるとき，環境変数PAGERに 設定されているページャーが使われる。何も設定されていないとき， moreが使われる。 なお，環境変数LANGにja，japanese，ja_JP.xxx が設定されていれば，日本語でヘルプが表示される。     
help;       // 関数一覧を表示
help eig;   // 組み込み関数eigに関する情報を表示
help Ode;   // ユーザ関数Odeに関する情報を表示
