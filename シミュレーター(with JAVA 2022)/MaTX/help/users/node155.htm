対角行列とブロック対角行列

対角行列及びブロック対角行列を定義する時， 全ての成分のクラスは一致しなければならない。 整数，実数，多項式等のスカラーと行列を混在させてブロック対角行列を記述する には，スカラーを1 x 1の行列として記述する。 
M1 = diag(1.0, 2.0, 3.0);        // 実対角行列
CM1 = diag((1,2), (3,4), (5,6)); // 複素対角行列
M4 = diag(M1, M2, M3);           // ブロック対角行列
M3 = diag(M1, M2, [1]);          // ブロック対角行列
