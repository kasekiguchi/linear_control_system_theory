有理多項式の係数のシフト

    有理多項式の係数を高次方向と低次方向にシフトするには， 関数higher()とlower()をそれぞれ用いる。 第2引数にシフトする回数を指定するが，回数を省略すると1回のシフトを意味する。 
  s = $;
  R1 = (s + 2)/(2*s + 3);
  R2 = higher(R1);        // 係数を高次方向に1回シフト
  R2 = higher(R1,2);      // 係数を高次方向に2回シフト
  R3 = lower(R2);         // 係数を低次方向に1回シフト
  R3 = lower(R2,2);       // 係数を低次方向に2回シフト
