ディレクトリを変更する

  作業ディレクトリを変更するには，コマンドchdirを使用する。 ただし，文字列中の ¥t や ¥n などは特殊文字として 解釈されるので，バックスラッシュ ¥ を重ねる必要がある。 
chdir "/usr/local/tmp"     // UNIX
chdir "c:/tmp"             // DOS(1)
chdir "c:¥¥tmp"            // DOS(2)
