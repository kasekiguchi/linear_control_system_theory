停止

  プログラムを停止するには，pauseコマンドを使用する。 停止する時間を指定することができる。 
pause;                         // 何かキー入力するまで停止
pause 5;                       // 5秒間停止
pause "Hit Any Key¥n";         // メッセージを表示して，
                               // 何かキー入力するまで停止
pause "five seconds¥n", 5;     // メッセージを表示して，
                               // 5秒間停止
