matxの実行時のメッセージの説明

Call nested too deeply
関数の呼び出しが深すぎた。

Can't find the function
宣言された関数の定義が見つからなかった。

Can't open file
fprintf()によるファイルのオープンが出来ない。

Can't open help file
ヘルプファイルをオープンできない。

Can't open temporary file for help
ヘルプのための一時ファイルをオープンできない。

Confliction of the argument class
関数の引数の型が定義と異なる。

Data is broken or too few
<-で読み込んだデータが壊れている。

Duplicate switch label
１つのswitch文に同じラベルが２度使われた。

'extern' is used in argument definition
extern宣言が関数の引数の宣言で使われた。

Function cann't be edited
editコマンドで関数を編集しようとした。

Function cann't be read
readコマンドで関数を読み込もうとした。

Function returns no value
関数が値を返していない。

Incorrect number of arguments
関数の引数の数が定義と異なる。

Name too long
変数名前の長さが長すぎる。

No help file is available
ヘルプファイルが見つからない。

switch too deep
switch文の入れ子が深すぎる。

Switch label too much
switch文のラベルが多すぎる。

Syntax error was found in string evaluation
関数eval()による文字列の評価において文法エラーが検出された。
