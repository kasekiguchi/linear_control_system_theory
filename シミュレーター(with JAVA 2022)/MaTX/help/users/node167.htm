有理多項式行列の各要素の係数のシフト

    有理多項式行列の各要素の係数を高次方向と低次方向にシフトするには，関数
higher()とlower()をそれぞれ用いる。 第2引数にシフトする回数を指定するが，回数を省略すると1回のシフトを意味する。 
CRM2 = higher(CRM1);   // CRM1の高次方向への1回シフト
CRM3 = higher(CRM1,2); // CRM1の高次方向への2回シフト
CRM4 = lower(CRM2);    // CRM2の低次方向への1回シフト
CRM5 = lower(CRM3,2);  // CRM3の低次方向への2回シフト
