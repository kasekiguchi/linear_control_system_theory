代入式

MATXにおける代入は，C言語の代入式と同様に式として処理される。 そして，代入式の評価順序は右から左である。 ただし，リストから複数個の成分を取り出すための代入，行列の成分の代入， 多項式の係数の代入，リストの成分の代入，は式ではなく文として処理される。 例えば， 
  Matrix A;
  Polynomial B;
  List C;

  A = Z(2);
  A(1,1) = 1.0;           // 文
  A(1,2) = [[1 2][3 4]];  // 文

  B = Polynomial(Z(1,4));
  B(1) = 3.0;             // 文

  C = makelist(3);
  C(1) = 1;               // 文

そして， 
  Matrix A, B, C;

  A = B = C = Z(2,3);    // 式

は正しいが， 
  Matrix A, B, C;
  Polynomial D;
  List LL;

  LL = {1, (1,2), [1 2]};   // 文
  {A, B, C} = {A, B} = LL;  // 文

は正しくない。
