多項式

実数と複素数を係数とする1変数の多項式の数式処理を行なうことができる。 今のところ，多項式の係数は，昇べきの順に横ベクトルとして保存されている。

  ［宣言］ 

  Polynomial   a, b, c;    // 実多項式
  CoPolynomial a2, b2, c2; // 複素多項式

［例］ 

  2*s^2 + 3*s + 4,  (1,2)*s^2 + (3,4)*s + (5,6)
