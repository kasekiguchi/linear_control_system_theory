行列成分の削除

代入式の左辺に，削除する成分を指定した行列を書き，右辺に0 x 0の行列を 用いれば，左辺の行列から行単位あるいは列単位で成分を削除できる。
M1(1,:) = []; M1の1行を削除する
M1(2:3,:) = []; M1の2〜3行を削除する
M1(:,1) = []; M1の1列を削除する
M1(:,2:3) = []; M1の2〜3列を削除する
M1(2:3) = []; M1の(1,2),(1,3)成分を削除する
  (1,i)成分の指定する時，1を省略できる。
