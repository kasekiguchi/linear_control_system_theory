変数の存在を調べる(matxのみ)

    ある変数が定義されているか調べるには，変数名を引数として関数exist()を 呼ぶ。局所変数として定義されている場合は，2，大域変数としてのみ 定義されている場合は，1，定義されていない場合は，0が返される。 
  i = exist("A");
  if (i == 0)
    print "A dosen't exist.";
  } else if (i == 1) {
    print "A is a global variable.";
  } else if (i == 2) {
    print "A is a local variable.";
  }
