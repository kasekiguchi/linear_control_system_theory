リスト成分の付加

リストに新しい成分を付加するには関数appendlist()を使うか， +オペレータを使う。   
 x = {sqrt(3.0), ((1.0+2.0)*4.0,:)};
 appendlist(x, "Hello");
 // x ---> {sqrt(3.0), ((1.0+2.0)*4.0,:), "Hello"}

 y = {sqrt(3.0), ((1.0+2.0)*4.0,:)};
 y = y + {"Hello"};
 // y ---> {sqrt(3.0), ((1.0+2.0)*4.0,:), "Hello"}
