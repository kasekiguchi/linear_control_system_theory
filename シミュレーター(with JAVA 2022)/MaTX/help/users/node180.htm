行列の比較と配列の比較

行列の正定性や2つの行列の比較は， 関係演算子 >, >=, <, <= を 使って調べることができる。配列を関係演算子を用いて比較した場合， 各成分毎の比較が行なわれ，結果が(0または1)を成分とする配列として返される。 この結果は，関数all()，all_col()，all_row()， any()，any_col()，any_row() の引数として利用できる。
