複素行列

複素行列の記述方法には，複素数を成分として記述する方法と 一対の実行列をコンマ ``,'' で区切って丸括弧 ( と ) で囲む方法がある。 実部または虚部だけを指定し，もう一方に * を指定して 複素行列を記述することもできる。   
CM3 = (M1,M2); // CM3 = M1 + M2*i;    (M1,M2:実行列)
CM3 = (M1,*);  // CM3 = M1 + Z(M1)*i; (M1:実行列)
CM3 = (*,M1);  // CM3 = Z(M1)*i + M1; (M1:実行列)

複素行列の実部と虚部は， 関数Re()とIm()を用いてとり出すことができる。     
M1 = Re((M1,M2));  // 実部行列 = Re(複素行列)
M2 = Im((M1,M2));  // 虚部行列 = Im(複素行列)
