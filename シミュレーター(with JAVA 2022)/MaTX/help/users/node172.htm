式行列の成分の取り出し

行列の式を四角ブラケット [ と ] で囲むことによって 変数行列と同様に成分を取り出すことができる。  
[M1+M2](1,1) M1+M2の(1,1)成分
[M1+M2](3:5) M1+M2の3-5番目成分からなるベクトル
[M1+M2](3:) M1+M2の3番目以降の成分からなるベクトル
[M1+M2](1,:) M1+M2の第1行ベクトル
[M1+M2](1:2,2:3) M1+M2の1-2行，2-3列のブロック成分
[M1+M2](2:3,4) M1+M2の2-3行，4列成分よりなる列ベクトル
[M1+M2](2:,4) M1+M2の2行以降，4列成分よりなる列ベクトル
