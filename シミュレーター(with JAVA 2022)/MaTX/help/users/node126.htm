多項式の記述方法

多項式は，多項式変数をドル記号 $ を用いて定義し，その変数を用いて記述する。 多項式の係数が整数の時，自動的に実数に変換される。 1次の多項式の変数名には，任意の名前を使うことができるが， 出力される多項式の変数名は，常に s である。 クラス変換関数Polynomial()とMatrix()を使って 横ベクトルと多項式を相互に変換できる。 多項式の係数は，横ベクトルの成分と昇べきの順に一致する。     
  s = $;                    // 1次の係数が1の1次多項式
  P1 = 2*s^2 + 3*s + 4;     // 2次の実多項式
  P2 = Polynomial([4 3 2]); // P2 = 2*s^2 + 3*s + 4
  CP1 = (1,2)*s + (3,4);    // 1次の複素多項式
