Windows95/NTでmgplotを使う

      Windows95/NTのgnuplotはパイプに対応してないので，mgplotは コマンドをファイルに書き込み，そのファイル名を引数としてgnuplotを 起動する。mgplotはgnuplotに渡した全てのコマンドを記録して いるが，グラフを描画する毎にgnuplotを起動し直さなければならない。 起動回数を減らすにはmgplot_hold()関数が役に立つ。この関数で ホールドされたウィンドウに対応するgnuplotは，ホールドが解除される まで起動されることはない。以下の例では，最終行でホールドが解除されて 初めて，gnuplotが起動される。 
  w = [-PI:PI];
  mgplot_hold(1, 1);
  mgplot_subplot(1, 2, 1, 1);
  mgplot(1, w, sin(w));
  mgplot_subplot(1, 2, 1, 2);
  mgplot(1, w, cos(w));
  mgplot_hold(1, 0);
