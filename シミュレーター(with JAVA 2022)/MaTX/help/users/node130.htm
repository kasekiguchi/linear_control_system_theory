多項式の微分と積分

    多項式の微分と積分は，関数derivative()とintegral() を用いる。 第2引数に求める導関数(derivative)あるいは不定積分(indefinite integral) の階数を指定するが，階数を省略すると1階の導関数または不定積分を求める。 
  s = $;
  P1 = 3*s^2 + 4*s + 5;
  P2 = integral(P1);     // 2次の実多項式の1階不定積分
  P2 = integral(P1,2);   // 2次の実多項式の2階不定積分
  P3 = derivative(P2);   // 2次の実多項式の1階導関数
  P3 = derivative(P2,2)  // 2次の実多項式の2階導関数
