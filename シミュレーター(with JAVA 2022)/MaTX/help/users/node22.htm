登録関数の表示(matxのみ)

      登録されている組み込み関数とユーザ定義関数 を表示するにはwhatコマンドを使用する。 
  what

また，whichコマンドを使えば，どのファイルで定義されたユーザ定義関数 が登録されているか分かる。 変数を指定しないでwhichコマンドを使うと，登録されている全ての 関数の情報が表示される。 
  which
  which bode, ss2tfm

whichコマンドで表示されるファイルは，require修飾子 3.1付きの関数宣言 で変更できる。
