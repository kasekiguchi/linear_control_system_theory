多項式行列の各要素の係数のシフト

    多項式行列の各要素の係数のを高次方向と低次方向にシフトするには，関数
higher()とlower()ををそれぞれ用いる。 第2引数にシフトする回数を指定するが，回数を省略すると1回のシフトを意味する。 
CPM2 = higher(CPM1);    // CPM1の高次方向への1回シフト
CPM3 = higher(CPM1,2);  // CPM1の高次方向への2回シフト
CPM4 = lower(CPM2);     // CPM2の低次方向への1回シフト
CPM5 = lower(CPM3,2);   // CPM3の低次方向への2回シフト
