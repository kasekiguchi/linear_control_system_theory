多項式の係数のシフト

    多項式の係数を高次方向と低次方向にシフトするには， 関数higher()とlower()をそれぞれ用いる。 第2引数にシフトする回数を指定するが，回数を省略すると1回のシフトを意味する。 
  s = $;
  P1 = 3*s^2 + 4*s + 5;
  P2 = higher(P1);    // 係数を高次方向に1回シフト
  P2 = higher(P1,2);  // 係数を高次方向に2回シフト
  P3 = lower(P2);     // 係数を低次方向に1回シフト
  P3 = lower(P2,2);   // 係数を低次方向に2回シフト
