商 (Division)

   整数の割算では小数部は切捨てられる。 行列と行列の商 ``X = A ¥ B''は，``A * X = B''の解で， ``X = A / B''は，``X * B = A''の解である。 Aが正方行列の時，LU分解を用いて解が求められ，Aが正方行列で ない時，最小二乗解が求められる。

OP3=OP1 / OP2 --> (OP3 = OP1 * OP2‾)
OP3=OP1 ¥ OP2 --> (OP3 = OP1‾ * OP2)

整数=整数 / 整数
実数=実数 / 実数
複素数=複素数 / 複素数
