リストの合成と取り出し

長いリストは，短いリストを成分として作ることができる。 リストxとリストyから長いリストzを作るには， 
  x = {4, (3,4), [1 2]};
  y = {"Hello", 2.0};
  z = {x, y};
  // z = {{4, (3,4), [1 2]}, {"Hello", 2.0}}

のように書く。 また，成分の範囲を指定することにより， リストの任意の部分を取り出すことができる。例えば， 
  x2 = x(1:2);                  // x2 = {4, (3,4)}

  x3 = {4, (3,4), [1 2]}(1:2);  // x3 = {4, (3,4)}

により，リストxから最初の2個の成分を取り出すことができる。
