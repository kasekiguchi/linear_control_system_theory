リストの記述方法

リストを記述するには，成分となる値をコンマ ``,'' で区切って書き並べ，
ブレース { と } で囲む。 
  x = {4, (3,4), [1 2]};

により，成分が整数，複素数，行列であるリストxに代入される。 リストの成分には任意の式を書くことができる。 
  x = {sqrt(3.0), ((1.0+2.0)*4.0,:), [1 2] + [3 4]};

により，成分が実数，複素数，行列であるリストがxに代入される。関数
makelist()を使って，任意個の成分を持つリストを 定義することができる。   この時，リストの成分は未定義(UNDEF)である。  
  x = makelist(3);
