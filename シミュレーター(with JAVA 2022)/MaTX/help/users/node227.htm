現在，RTMATXが動作する環境は，次の通りである。 今のところ，リアルタイムプログラムはインタプリタ上では動作しない。
  • IBM PC互換機のMS-DOS上で，DJGPP 2.01 21.2を用いる。
  • PC9801シリーズのMS-DOS上で，Borland C++ 3.1を用いる。
  • IBM PC互換機のMS-DOS上で，Borland C++ 3.1を用いる。
今後，動作環境を拡張して行く予定である。
