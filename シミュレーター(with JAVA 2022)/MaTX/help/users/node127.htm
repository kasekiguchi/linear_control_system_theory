複素多項式

複素多項式の入力方法には，複素数を係数として記述する方法と 一対の実多項式をコンマで ``,'' 区切って丸括弧 ( と )で 囲む方法がある。 実部または虚部だけを指定し，もう一方に * を指定して 複素多項式を記述することもできる。   
  CP1 = (P1,P2);  // CP1 = P1 + P2*i; (P1, P2: 実多項式)
  CP1 = (P1,*);   // CP1 = P1 + 0*i;  (P1: 実多項式)
  CP1 = (*,P1);   // CP1 = 0  + P1*i; (P1: 実多項式)

複素多項式の実部と虚部は，関数Re()とIm() を用いて取り出すことができる。     
  P1 = Re((P1,P2));  // 実部多項式 = Re(複素多項式)
  P2 = Im((P1,P2));  // 虚部多項式 = Im(複素多項式)
