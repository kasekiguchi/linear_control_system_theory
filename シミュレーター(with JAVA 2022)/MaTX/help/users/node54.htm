行列データのファイル入出力

         行列をファイルへ書き込むには，printコマンドを用い 出力先を記号>>によってファイルにリダイレクトする。 ファイル名の拡張子は，自由に決めることができるが，matを用いることが 推奨される。行列データは，MATデータフォーマットでファイルに書き込まれる。   
  print A >> "filename.mat";

行列をファイルから読み込むには，readコマンドを用い 入力先を記号<<によってファイルにリダイレクトする。 ファイル名の拡張子は，自由に決めることができるが，matを用いることが 推奨される。     
  read A << "filename.mat";
