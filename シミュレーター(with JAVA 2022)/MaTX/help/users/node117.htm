文字列

任意個の文字の列を二重引用符 " と " で囲んで記述する。 タブと改行はそれぞれ ¥t と ¥n で表す。 + 演算子で文字列を結合できる。

［宣言］ 

String a, b, c;

［例］ 

"FileName", "File¥tName¥n"
"Hello" + " " + "world"             // "Hello world"
