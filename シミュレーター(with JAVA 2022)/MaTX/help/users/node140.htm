有理多項式の微分

  有理多項式の微分は，関数derivative()を用いる。 有理多項式の積分は，今のところ実現されていない。 第2引数に求める導関数の階数を指定するが，階数を省略すると1階の導関数を求める。 
  s = $;
  R1 = (s + 2)/(2*s + 3);
  R2 = derivative(R1);       // 実有理多項式の1階導関数
  R2 = derivative(R1,2);     // 実有理多項式の2階導関数
