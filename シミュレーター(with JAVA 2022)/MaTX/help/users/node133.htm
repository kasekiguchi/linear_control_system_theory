多項式関数

多項式関数の一覧を以下に示す。

                            

Polynomial P1;
Integer i1;
Real tol;

Matrix  roots(P1);          多項式の根
Integer degree(P1);         多項式の次数
Integer isreal(P1);         実多項式の有無
                            (真 = 1, 偽 = 0)    
Integer iscomplex(P1);      複素多項式の有無
                            (真 = 1, 偽 = 0)

Polynomial floor(P1)        係数の最低値の整数への丸め
Polynomial ceil(P1)         係数の最高値の整数への丸め
Polynomial fix(P1)          係数の 0 方向の整数への丸め
Polynomial round(P1)        係数の最も近い整数への丸め
Polynomial round2z(P1)      係数の 0 への丸め(tol = EPS)
Polynomial round2z(P1, tol) 係数の絶対値がtolより小なら0
Polynomial pow(P1, i1)      P1^i1
Polynomial trans(P1)        P1'
Polynomial conj(P1)         P1 の共役
Polynomial conjtrans(P1)    P1#
Rational   inv(P1)          P1‾
