一般データのファイル入出力

        データをファイルへ書き込むには，printコマンドを用いる。 出力先を記号->によってファイルにリダイレクトする。 matxの場合，拡張子を省略すると，拡張子はmxとなる。 データは，MXデータフォーマットでファイルに書き込まれる。     
  print a, b, c, d -> "filename.mx";

データをファイルからの読み込むには，readコマンドを用いる。 入力先を記号<-によってファイルにリダイレクトする。 matxの場合，拡張子を省略すると，拡張子はmxとなる。 matxの場合，ファイル名だけを指定すると，ファイルに保存されている 全てのデータが読み込まれる。ただし，保存したデータと同じ変数に読み込まれる。     
  read a, b, c, d <- "filename.mx";

この方法はデータをバイナリ形式でファイルに保存するので， データの精度が保存される。次に示すように入力先を変更する記号 <-は式の中に現われる可能性がある。 
  print 0<-a;

この式は-aが正である(0より大きい)かどうか調べる意図で書かれているが， < と - の間にスペースが無いので，まとめて<-として解釈され， 文法エラーになる。< と- の間にスペースを入れると 意図通りに解釈される。
