ウィンドウ・マネ−ジャ−の設定

      Xウィンドウ・システムのウィンドウ・マネ−ジャ−としてtwm を使う場合，以下の記述を各自のウィンドウ・マネ−ジャ−の設定ファイル .twmrcに加えることによって，xplotを使用するときに， キャンバス・ウィンドウだけが表示されるようにできる。 また，メインウィンドウやデ−タ管理ウィンドウなどは，アイコン・ マネ−ジャによって自由に操作することができる。 
###############################################
#       ICON MANAGER
###############################################

SortIconManager # keep them in alphabetic order

IconManagers
{
   "xplot"   "=280x900+2+2" 2
}

IconManagerShow
{
   "xplot"
}

IconifyByUnmapping
{
  "xplot"
}

StartIconified
{
        "XPlot"
        "2D_Info"
        "SemiLog_Info"
        "d1plot-data-1"
        "d1plot-data-2"
        "d1plot-data-3"
        "d2plot-data-1"
        "d2plot-data-2"
        "d2plot-data-3"
        "splot-data-1"
        "splot-data-2"
        "splot-data-3"
}
#############################################
