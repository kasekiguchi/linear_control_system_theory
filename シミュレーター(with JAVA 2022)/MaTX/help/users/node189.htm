リスト関数

リスト関数の一覧を以下に示す。 関数fscanf()の第1引数は，fopen()でオープンした ファイルのファイルディスクリプタを用いる。

          

Integer length(List L1);               リストの成分の数
List    appendlist(List L1, Object A); L1に成分Aを付加する。
List    fscanf(Integer fd, String format, ...);  ファイル入力
