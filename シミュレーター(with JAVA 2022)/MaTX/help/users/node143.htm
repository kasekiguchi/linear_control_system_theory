有理多項式関数

有理多項式関数の一覧を以下に示す。

                            

Rational R1;
Integer i1;
Real tol;

Matrix  zeros(R1);          有理多項式の零点
Matrix  poles(R1);          有理多項式の極
Integer isreal(R1);         実有理多項式の有無
                            (真 = 1, 偽 = 0)
Integer iscomplex(R1);      複素有理多項式の有無
                            (真 = 1, 偽 = 0)

Rational floor(R1)          係数の最低値の整数への丸め
Rational ceil(R1)           係数の最高値の整数への丸め
Rational fix(R1)            係数の 0 方向の整数への丸め
Rational round(R1)          係数の最も近い整数への丸め
Rational round2z(R1)        係数の0への丸め(tol = EPS)
Rational round2z(R1, tol)   係数の絶対値がtolより小なら0
Rational pow(R1, i1)        R1^i1
Rational trans(R1)          R1'
Rational conj(R1)           R1 の共役
Rational conjtrans(R1)      R1#
Rational inv(R1)            R1‾
