多項式の係数の参照

多項式の名前の次に次数を丸括弧 ( と ) で囲むことによって 多項式の係数を参照することができる。 ただし，複素多項式の係数には実数と複素数を代入できるが，実多項式の係数には 実数しか代入できない。 また，定義されている次数以上の係数を参照することはできない。 
  r1 = P1(1);   // 実多項式P1の1次の係数
  c1 = CP1(2);  // 複素多項式CP1の1次の係数
  P1(0) = r2;   // 実数r1を多項式P1の0次の係数に代入
  CP1(3) = c2;  // 複素数c1を多項式CP1の3次の係数に代入
