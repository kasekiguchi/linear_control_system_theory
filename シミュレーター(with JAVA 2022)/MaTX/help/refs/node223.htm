std

【目的】
std - 全ての成分の標準偏差
【形式】 
x = std(X)
   (Real  |Complex |Polynomial|Rational) x;
   (Matrix|CoMatrix|PoMatrix  |RaMatrix) X;
  
x = std(X)
   (Real |Complex|Polynomial|Rational) x
   (Array|CoArray|PoArray   |RaArray ) X;
  
x = std(X)
   Integer x;
   Index X;
【詳細】
std(X)は，全ての成分の標準偏差を求める。
【例題】 
>> x = std([1 2 3 4 5])
x = 1.41421
【参照】
sum(2.228), median(), mean(2.130), std_row(2.226), std_col(2.225)
