all_row

【目的】
all_row - 論理演算関数(行毎に全ての成分が 0 でないか判定)
【形式】 
x = all_row(X)
   Array x;
   (Matrix|Array|Index) X;
【詳細】
all_row(X) は，X の行毎に全ての成分が 0 でない場合に 1 を 1 つも 0 の成分があれば 0 を割り当て，0 と 1 からなる列ベク トル(配列)を返す。
【例題】
行列型のデータ A の行毎に，全ての成分が 5 より大きいか判定する。
all_row(A .> 5)
【参照】
all(2.5), all_col(2.6), any(2.8), any_row(2.10), any_col(2.9)
