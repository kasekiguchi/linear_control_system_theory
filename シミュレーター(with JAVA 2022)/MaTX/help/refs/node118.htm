lu

【目的】
lu - LU分解
【形式】 
{L,U} = lu(A)
   Matrix L,U;
   Matrix A;
【詳細】
{L,U} =lu(A)は，行列 A のLU分解を求める。この分解は，正方行列 A を下三角行列を行置換した行列 L と上三角行列 U の積に分解する。 これらの行列の間には，A = L * U の関係がある。
【算法】
文献 1 の算法を使う。
【例題】 
>> A = [[1 2 3][6 5 4][7 8 1]];
>> {L, U} = lu(A);
>> print L, U
=== [L] : (  3,  3) ===

(  2)  8.57142857E-01  1.00000000E+00  0.00000000E+00
(  3)  1.00000000E+00  0.00000000E+00  0.00000000E+00
