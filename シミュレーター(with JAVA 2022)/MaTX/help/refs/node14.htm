any_row

【目的】
any_row - 論理演算関数(行毎に 0 でない成分が存在するか判定)
【形式】 
x = any_row(X)
   Array x;
   (Matrix|Array|Index) X;
【詳細】
any_row(X) は，X の行毎に 0 でない成分が存在する場合 1 を， 全ての成分が 0 の場合 0 を割り当て，0 と 1 からなる列ベク トル(配列)を返す。
【例題】
行列型のデータ A の各行に 5 より大きい成分が存在するか判定する。
any_row(A .> 5)
【参照】
all(2.5), all_row(2.7), all_col(2.6), any(2.8), any_col(2.9)
