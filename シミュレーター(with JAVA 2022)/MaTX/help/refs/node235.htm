trace

【目的】
trace - トレース(対角成分の和)
【形式】 
x = trace(X)         c = trace(X)
   Real x;              Complex c;
   Matrix X;            CoMatrix X;
  
p = trace(X)         r = trace(X)
   Polynomial p;        Rational r;
   PoMatrix X;          RaMatrix X;
【詳細】
trace(X)は行列 X のトレース(対角成分の和)を求める。X が実行列の とき実数を，X が複素行列のとき複素数を，X が多項式行列のとき 多項式を，有理多項式行列のとき有理多項式を返す。
【例題】 
>> X = [[1 2 3][4 5 6][7 8 9]];
>> x = trace(X)
x = 15
