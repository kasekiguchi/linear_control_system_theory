asinh

【目的】
asinh - 双曲(線)逆正弦
【形式】 
y = asinh(x)                     Y = asinh(X)
   (Real   |Real|Complex) y;        Array Y;
   (Integer|Real|Complex) x;        Array X;
【詳細】
asinh(x)は，スカラ型(整数, 実数, 複素数) x の逆双曲(線)正弦を求める。 角度はラジアンで計算される。 asinh(X)は，配列 X の各成分の逆双曲(線)正弦からなる配列 Y を求める。 Y の大きさは X の大きさと同じ。
【算法】
asinh(z) = log(z + sqrt(z*z + 1))
【例題】 
>> y = 0.481212
y = 0.481212
【参照】
acos(2.3), asin(2.12), atan(2.14), acosh(2.4), atanh(2.16)
