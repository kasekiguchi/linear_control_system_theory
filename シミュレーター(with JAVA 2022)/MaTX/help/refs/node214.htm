sinh

【目的】
sinh - 双曲(線)正弦
【形式】 
y = sinh(x)                      Y = sinh(X)
   (Real   |Real|Complex) y;        Array Y;
   (Integer|Real|Complex) x;        Array X;
【詳細】
sinh(x)は，スカラ型(整数, 実数, 複素数) x の双曲(線)正弦を求める。 角度はラジアンで計算される。 sinh(X)は，配列 X の各成分の双曲(線)正弦からなる配列 Y を求める。 Y の大きさは X の大きさと同じ。
【算法】 
i = (0,1);
sinh(z) = (exp(-z) - exp(z))/2;
【例題】 
>> y = sinh(0.5)
y = 0.521095
範囲 のsinh(x)のグラフを表示する。
>> x = [-PI:0.05:PI];
>> mgplot(1, x, sinh(x), {"sinh(x)"});
【参照】
sin(2.211), cosh(2.37), tanh(2.235), asinh(2.13), exp(2.61)
