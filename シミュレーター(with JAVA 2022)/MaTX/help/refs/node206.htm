shiftDown

【目的】
shiftDown - 全ての行を下方向へシフトする
【形式】 
Y = shiftDown(X)               Y = shiftDown(X,k)
   (Matrix|Array|Index) Y;        (Matrix|Array|Index) Y;
   (Matrix|Array|Index) X;        (Matrix|Array|Index) X;
                                  Integer k;
【詳細】
shiftDown(X)は，行列型(行列|配列|指数) X の全ての行を下方向へ 1 行シフトした行列を作る。一番上の行は，ゼロになる。 shiftDown(X,k)は，下方向へ k 行シフトする。
【例題】 
>> Y = shiftDown([[1 2][3 4][5 6]])
=== [Y] : (  3,  2) ===
           (  1)           (  2)     
(  1)  0.00000000E+00  0.00000000E+00
(  2)  1.00000000E+00  2.00000000E+00
(  3)  3.00000000E+00  4.00000000E+00
【参照】
rotateDown(2.184), shiftUp(2.210), shiftLeft(2.208), shiftRight(2.209)
