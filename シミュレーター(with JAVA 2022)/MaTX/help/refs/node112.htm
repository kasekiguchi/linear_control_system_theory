linspace

【目的】
linspace - 等間隔ベクトル
【形式】 
y = linspace(d1,d2)
   Array y;
   Real d1,d2;

y = linspace(d1,d2,n)
   Array y;
   Real d1,d2;
   Integer n;
【詳細】
linspace(d1,d2) は，d1 と d2 を等間隔で 100 分割した 100 個の 値からなる行ベクトルを返す。 linspace(d1,d2,N) は，d1 と d2 を等間隔で N 分割した N 個の 値からなる行ベクトルを返す。
【例題】 
>> y = linspace(0.0,PI,4)
 === [y] : (  1,  4) ===
            (  1)         (  2)         (  3)         (  4)     
 (  1)  0.000000E+00  1.047197E+00  2.094395E+00  3.141592E+00
