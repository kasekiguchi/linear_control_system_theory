roots

【目的】
roots - 多項式の根
【形式】 
X = roots(p)
   CoMatrix X;
   Polynomial p;
【詳細】
roots(P)は，多項式 p = 0 の根を求める。
【例題】 
>> s = Polynomial("s");
>> p = s^2 + 2*s + 3; 
>> X = roots(p)
=== [X] : (  2,  1) ===
       [ (  1)-Real      (  1)-Imag ]
(  1) -1.00000000E+00  1.41421356E+00
(  2) -1.00000000E+00 -1.41421356E+00
【参照】
zeros(2.249), poles(2.158), eigval(2.54)
