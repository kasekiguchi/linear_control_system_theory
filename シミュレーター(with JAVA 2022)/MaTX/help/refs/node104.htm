isempty

【目的】
isempty - 空行列の判定
【形式】 
i = isempty(X)
   Integer i;
   Matrix X;
【詳細】
X が空行列のとき，isempty(X) は 1 を，そうでないとき，0 を 返す。空行列とは，0 * 0 の行列である。
【例題】 
>> i = isempty([])
i = 1
>> i = isempty([1])
i = 0
【参照】
isnan(2.108), isinf(2.107)
