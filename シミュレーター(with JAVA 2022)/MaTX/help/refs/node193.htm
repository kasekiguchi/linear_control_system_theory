rtIsTimeOut

【目的】
rtIsTimeOut - リアルタイム処理のオーバーランの判定
【形式】 
i = rtIsTimeOut()
   Integer i;
【詳細】
rtIsTimeOut()は，リアルタイム処理がサンプリング時間内に終了 しなったか場合は，1 を，そうでなければ，0 を返す。
【例題】 
if (rtIsTimeOut()) {
    print "Time out¥n";
}

rtIsRunning(2.193), rtIsOnLine(2.191), rtIsRehearsal(2.192)
