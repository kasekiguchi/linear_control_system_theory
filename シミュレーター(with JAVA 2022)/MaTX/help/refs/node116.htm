logspace

【目的】
logspace - 対数等間隔ベクトル
【形式】 
y = logspace(d1,d2)
   Array y;
   Real d1,d2;

y = logspace(d1,d2,n)
   Array y;
   Real d1,d2;
   Integer n;
【詳細】
logspace(d1,d2) は，10d1 と 10d2 を対数等間隔で 50 分割した 50 個 の値からなる行ベクトルを返す。もし，d2 が PI なら，10d1 と を 対数等間隔する。 logspace(d1,d2,N) は，10d1 と 10d2 を対数等間隔で N 分割した N 個の 値からなる行ベクトルを返す。
【例題】 
>> y = logspace(-2.0,2.0,4)
 === [y] : (  1,  4) ===
            (  1)         (  2)         (  3)         (  4)     
 (  1)  1.000000E-02  2.154434E-01  4.641588E+00  1.000000E+02
