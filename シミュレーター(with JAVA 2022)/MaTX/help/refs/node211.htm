singleftvec

【目的】
singleftvec - 特異値分解の左変換行列
【形式】 
U = singleftvec(A)
   Matrix U;
   Matrix A;
【詳細】
singleftvec(A)は，行列 A の特異値分解の左側の変換行列を求める。 変換行列 U は，直交行列(ユニタリ行列)である。
【算法】
文献 1 の算法を使う。
【注意】
繰り返し計算で特異値が求まらない場合は，次のメッセージが表示される。

Solution will not converge

【例題】 
>> A = [[1 2 3][4 5 6]];
>> U = singleftvec(A)
=== [U] : (  2,  2) ===
           (  1)           (  2)
(  1) -3.86317703E-01 -9.22365780E-01
(  2) -9.22365780E-01  3.86317703E-01
【参照】
svd(2.231), singval(2.214), singrightvec(2.213), maxsing(2.127), minsing(2.138)

1.
