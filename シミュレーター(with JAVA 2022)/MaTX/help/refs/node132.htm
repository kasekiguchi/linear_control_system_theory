menu

【目的】
menu - メニュー
【形式】 
i = menu(L)       i = menu(L,k)
   Integer i;        Integer i;
   List L;           List L;
                     Integer k;
【詳細】
menu(L)は，メニューを表示し，選択された項目の番号を返す。 リスト L は，メニューのタイトル(文字列)，項目(文字列)を 成分にもつ。 L の構成を次に示す。
  
    L = {"title", "item1", "item2", ..., "itemn"};
menu(L,k)は，メニューの選択番号の初期値を k に設定する。
【例題】 
L = {"title", "item1", "item2", ..., "itemn"};
i = menu(L);
