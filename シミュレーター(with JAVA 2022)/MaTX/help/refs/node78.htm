frobnorm

【目的】
frobnorm - フロベニウスノルム
【形式】 
x = frobnorm(X)
   Real x;
   (Matrix|Array) X;
【詳細】
frobnorm(X)は，全ての成分のフロベニウスノルムを求める。
【例題】 
>> x = frobnorm([[1 2 3][4 5 6]])
x = 9.53939
【参照】
frobnorm_row(2.81), frobnorm_col(2.80), infnorm(2.99), norm(2.141)
