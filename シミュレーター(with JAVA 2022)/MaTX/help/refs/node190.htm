rtIsOnLine

【目的】
rtIsOnLine - リアルタイム処理中の判定
【形式】 
i = rtIsOnLine()
   Integer i;
【詳細】
rtIsOnLine()は，リアルタイム処理中(rtStart()が呼ばれてから rtStop()が呼ばれるまでの間)なら 1 を，そうでなければ，0 を 返す。
【例題】 
if (rtIsOnLine()) {
    print "Now, on line¥n";
}
