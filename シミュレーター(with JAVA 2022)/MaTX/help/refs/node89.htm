help

【目的】
help - ヘルプメッセージ
【形式】 
help;
  
help keyword;
【詳細】
インタプリタのコマンドラインで help と入力すればコマンドと 関数の一覧が表示される。初めに組み込み関数 (MATX内部にC言語で 実装された関数)，次にユーザ関数 (MATX言語で記述された関数)が アルファベット順に表示される。これらの名前を help の後ろに 関数名やコマンド名を付けると，詳しい説明を見ることができる help は，インタプリタでしか使用できない。
【例題】 
help eig

who(2.246), whos(2.247), which(2.244), what(2.243)
