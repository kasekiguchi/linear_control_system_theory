atanh

【目的】
atanh - 双曲(線)逆正接
【形式】 
y = atanh(x)                     Y = atanh(X)
   (Real   |Real|Complex) y;        Array Y;
   (Integer|Real|Complex) x;        Array X;
【詳細】
atanh(x)は，スカラ型(整数, 実数, 複素数) x の逆双曲(線)正接を求める。 角度はラジアンで計算される。 atanh(X)は，配列 X の各成分の逆双曲(線)正接からなる配列 Y を求める。 Y の大きさは X の大きさと同じ。
【算法】
atanh(z) = 1/(2 * log((1+z)/(1+z)))
【例題】 
>> y = atanh(0.5)
y = 0.549306
【参照】
asin(2.12), acos(2.3), atan(2.14), asinh(2.13), acosh(2.4)
