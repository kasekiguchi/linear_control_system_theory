norm

【目的】
norm - ノルム
【形式】 
x = norm(A)              x = norm(A,k)
   Real x;                  Real x;
   (Matrix|Array) A;        (Matrix|Array) A;
                            Integer k;
【詳細】
norm(A)は，行列 A の最大特異値を求める。norm(A,1) は，A の 1-ノルム(列毎の和の最大値)を求める。norm(A,2)は，norm(A)と 等しい。
【例題】 
>> x = norm([[1 2 3][4 5 6]])
x = 9.50803
>> x = norm([[1 2 3][4 5 6]],1)
x = 9
>> x = norm([[1 2 3][4 5 6]],2)
x = 9.50803
【参照】
frobnorm(2.79), infnorm(2.99)
