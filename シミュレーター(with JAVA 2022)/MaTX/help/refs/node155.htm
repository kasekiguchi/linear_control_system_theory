pause

【目的】
pause - プログラムを停止する
【形式】 
pause;
  
pause message;
    String message;
  
pause time;
    Real time;
  
pause message, time;
    String message;
    Real time;
【詳細】
pause を引数無しで使用すると，何かキーが入力されるまで停止する。 文字列を引数にすると，その文字列が表示される。停止する時間(秒)で 指定できる。
【例題】 
pause;
pause "Hit Any Key¥n";
【参照】
getch(2.85)
