strchr

【目的】
strchr - 文字列中の文字の位置
【形式】 
k = strchr(str,c)
   Integer i;
   String str,c;
【詳細】
strchr(str,c)は，文字列 str 中に文字 c が存在すれば，その位置を， 存在しなければゼロを返す。
【例題】 
>> S = "Hello world";
>> i = strchr(S, "w")
i = 7
>> S(i) = "W";
>> print S
S = Hello World
>> j = strchr(S, "A")
j = 0
【参照】
find(2.70)
