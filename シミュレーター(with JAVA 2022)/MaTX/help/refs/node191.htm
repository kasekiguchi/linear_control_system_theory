rtIsRehearsal

【目的】
rtIsRehearsal - リアルタイム処理のリハーサル中の判定
【形式】 
i = rtIsRehearsal()
   Integer i;
【詳細】
rtIsRehearsal()は，リアルタイム処理のリハーサル中 なら 1 を，そうでなければ，0 を返す。 リハーサルは，rtStartRehearsal()が呼ばれてから rtStopRehearsal()が 呼ばれるまでの間である。
【例題】 
if (rtIsRehearsal()) {
    print "Now, in the rehearsal¥n";
}
【参照】
rtIsRunning(2.193), rtIsOnLine(2.191)
