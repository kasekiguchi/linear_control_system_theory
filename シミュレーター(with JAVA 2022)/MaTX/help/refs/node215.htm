size

【目的】
size - 行列の行数と列数
【形式】 
{m,n} = size(X)
   Integer m,n;
   (Matrix|Array|Index) X;
【詳細】
size(X)は(行列|配列|指数) X の行数 m と列数 n を返す。
【例題】 
>> X = [[1 2 3][4 5 6]];
>> {m,n} = size(X);
【参照】
Rows(2.190), Cols(2.32), length(2.112)
