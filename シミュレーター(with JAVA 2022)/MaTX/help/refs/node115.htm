log10

【目的】
log10 - 常用対数
【形式】 
y = log10(x)                     Y = log10(X)
   (Real   |Real|Complex) y;        Array Y;
   (Integer|Real|Complex) x;        Array X;
【詳細】
log10(x)は，スカラ型(整数, 実数, 複素数) x の常用対数を求める。 log10(X)は，配列 X の各成分の常用対数からなる配列 Y を求める。 Y の大きさは X の大きさと同じになる。
【例題】 
>> y = log10(100)
y = 2
【参照】
sin(2.211), cos(2.36), tan(2.234), log(2.115), exp(2.61)
