ONE

【目的】
ONE - 1 で満たされた行列
【形式】 
Y = ONE(n)        Y = ONE(m,n)
   Matrix Y;         Matrix Y;
   Integer n;        Integer m,n;
  
Y = ONE(X)        Y = ONE(m,n,X)
   Matrix Y;         Matrix Y;
   Integer X;        Integer m,n;  
                     Matrix X;
【詳細】
整数 n について，ONE(n)は n * n の 1 で満たされた行列を返す。 ONE(m,n) は，m * n の 1 で満たされた行列を返す。X が行列のとき， ONE(X) は，X と同じ大きさの 1 で満たされた行列を返す。ONE(m,n,X) は，大きさが m*Rows(X) * n*Cols(X)の 1 で満たされた行列を返す。
【例題】 
>> Y = ONE(2)
=== [Y] : (  2,  2) ===
           (  1)           (  2)     
(  1)  1.00000000E+00  1.00000000E+00
(  2)  1.00000000E+00  1.00000000E+00
>> Y = ONE(2,3)
=== [Y] : (  2,  3) ===
           (  1)           (  2)           (  3)     
(  1)  1.00000000E+00  1.00000000E+00  1.00000000E+00
(  2)  1.00000000E+00  1.00000000E+00  1.00000000E+00
【参照】
Z(2.248), I(2.93)
