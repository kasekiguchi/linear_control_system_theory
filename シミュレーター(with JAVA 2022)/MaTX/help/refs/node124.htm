maximum_col

【目的】
maximum_col - 列毎の最大値とその指数
【形式】 
{Y,idx} = maximum_col(X)
   (Matrix|Array|Index) Y;
   Index idx;
   Index X;
【詳細】
X が(行列|配列|指数)のとき，{Y,idx} = maximum_col(X) は列毎に成分 の最大値をを求め，行ベクトル Y で返す。また，最大値の成分の番号か らなる指数 idx を返す。 X が縦ベクトルまたは横ベクトルのとき，maximum_col(X) は，ベクトル の成分の最大値とその成分の番号を返す。
【例題】 
>> {Y, idx} = maximum_col([[1.2, -2.3][3.1, 0]]) 
>> print Y, idx;
=== [Y] : (  1,  2) ===
           (  1)           (  2)     
(  1)  3.10000000E+00  0.00000000E+00
=== [idx] : (  1,  2) ===
           (  1)           (  2)     
(  1)  2.00000000E+00  2.00000000E+00
>> {Y, idx} = maximum_col([(1,2), (3,4)])
>> print Y, idx;
=== [Y] : (  1,  1) ===
       [ (  1)-Real      (  1)-Imag ]
(  1)  3.00000000E+00  4.00000000E+00
=== [idx] : (  1,  1) ===
           (  1)     
(  1)  2.00000000E+00
