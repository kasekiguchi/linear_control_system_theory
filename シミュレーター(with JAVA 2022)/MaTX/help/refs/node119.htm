lu_p

【目的】
lu_p - 並び替え付きLU分解
【形式】 
{L,U,P} = lu_p(A)
   Matrix L,U,P;
   Matrix A;
【詳細】
{L,U,P} = lu_p(A)は，行列 A の並び替え付きLU分解を求める。この 分解は，正方行列 Aを下三角行列 L，上三角行列 U，置換行列 P の積 に分解する。これらの行列の間には，P * A = L * U の関係がある。
【算法】
文献 1 の算法を使う。
【例題】 
>> A = [[1 2 3][6 5 4][7 8 1]];
>> {L, U, P} = lu_p(A);
>> print L, U, P;
