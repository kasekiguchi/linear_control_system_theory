cumprod_row

【目的】
cumprod_row - 行毎の累積積
【形式】 
Y = cumprod_row(X)
   (Matrix|Array|Index) Y;
   (Matrix|Array|Index) X;
【詳細】
cumprod_row(X)は，(行列|配列|指数)の行毎の累積積を求め， (行列|配列|指数) X と同じ大きさの(行列|配列|指数) Y を返す。
【例題】 
>> Y = cumprod_row([[1 2][3 4]])
=== [Y] : (  2,  2) ===
           (  1)           (  2)     
(  1)  1.00000000E+00  2.00000000E+00
(  2)  3.00000000E+00  1.20000000E+01
【参照】
cumprod(2.38), cumprod_col(2.39)
