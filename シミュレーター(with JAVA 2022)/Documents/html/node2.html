動作環境

本プログラムでは，以下に示すプログラムを利用している．
  1. Java Runtime Environment
    本プログラムは JDK(Java 2 Development Kit) 1.2.2 の環境で開発された ものである．実行するためには，JRE(Java 2 Runtime Environment) 1.2.2 以上の環境を必要とする． JRE 及び JDK に関する情報およびソフトウェアのダウンロードは，Sun Microsystems (株)のWEBページ
    http://jp.sun.com/java
    から参照することができる．
  2. MaTX
    MaTX は科学や工学に必要な数値および数式計算をサポートする記述性 に優れたプログラミング言語である．東京工業大学にて開発され，主に システム理論(制御系の解析・設計・シミュレーション)や信号処理の研 究に使われてきた．文法のうち制御の流れや関数などのプログラム構造 はC言語とほぼ同じである．行列の演算や成分の操作法は市販の計算ソフ トウェアMATLABとほぼ同じである． このプログラムに関する情報およびダウンロードは，MaTX のWEBページ
    http://www.matx.org
    から参照することができる．
    (MaTX 本体は，著作権を放棄していないフリーソフトウェアである．)

配布するCD-ROM内には，Windows, Solaris(Sparc, Intel), Linux(Intel)用の， シミュレータプログラム，JRE および MaTX 実行ファイルをまとめた アーカイブ(LHAもしくはTAR+GZIP)を用意しているので，このアーカイブファイルを ハードディスク上で展開することで，シミュレーションを実行することができる．

本シミュレータは Java 言語に基づいて開発されているため，様々なオペレーティ ングシステムで実行することがでる． しかし，数値シミュレーションに MaTX という外部プログラムを使用している ため，外部プログラムは適当なOSに調整する必要がある．
